# Camera OS - Interface de Celular com Câmera e Scanner

## Sobre o projeto

O **Camera OS** é uma simulação de interface de smartphone desenvolvida com HTML, CSS e JavaScript.
O projeto simula um sistema de celular com câmera funcional (simulação), digitalização de documentos, login e interações dinâmicas.

---

 Objetivo

Criar uma interface moderna e interativa utilizando:
- Manipulação do DOM
- Eventos em JavaScript
- Validação de login
- Slideshow de imagens
- Estrutura semântica em HTML
- Layout com Flexbox

---

 Funcionalidades

-  Interface estilo smartphone 
- Tela de login com validação
- Sistema de câmera (simulação)
- Captura de foto (modo paisagem)
- Modo digitalização de documentos
- Filtros de imagem
-  Galeria com slideshow automático
-  Alertas e prompts interativos
-  Troca dinâmica de telas
- Manipulação de elementos com JavaScript
